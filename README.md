# ?? TradeOS Bootcamp - Полный 6-недельный курс

## Описание

Полноценный bootcamp для обучения full-stack разработке с нуля до production-ready разработчика.

## Структура курса

1. **Неделя 1:** Основы FastAPI и REST API
2. **Неделя 2:** Базы данных PostgreSQL + SQLAlchemy
3. **Неделя 3:** Аутентификация и авторизация (JWT)
4. **Неделя 4:** Интеграция с внешними сервисами
5. **Неделя 5:** Продвинутый фронтенд (React + TypeScript)
6. **Неделя 6:** DevOps и деплой в production

## Технологический стек

- **Backend:** Python, FastAPI, PostgreSQL, Redis, Celery
- **Frontend:** React, TypeScript, Redux, Tailwind CSS
- **DevOps:** Docker, GitHub Actions, Nginx, Kubernetes
- **Интеграции:** 1С API, Telegram Bot API, WebSocket, SMTP

## Как начать

1. Клонируйте репозиторий
2. Перейдите в папку нужной недели
3. Следуйте инструкциям в README.md
4. Выполняйте задания из ASSIGNMENT.md
